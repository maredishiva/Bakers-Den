# 🍰 Bakers Den

A fully responsive bakery website built using **HTML, CSS, and JavaScript** that allows users to explore delicious baked items, view menus, and manage their shopping cart efficiently with dynamic JavaScript functionalities.

## 🚀 Features

- 🏠 **Home Page:** Welcoming landing page with bakery highlights and product showcases.
- 👨‍🍳 **About Page:** Details about the bakery’s story, mission, and specialties.  
- 🍩 **Menu Page:** Displays bakery items with prices, descriptions, and “Add to Cart” buttons.  
- 📞 **Contact Page:** Users can reach out via a clean and simple contact form.  
- 🛒 **Cart Functionality:**  
  - Add items to the cart dynamically.  
  - Update or remove items from the cart.  
  - Real-time cart total calculation.  
  - “Buy Now” button logic with confirmation functionality.

## 🕹️ How to Use
- Launch the index.html file to open the website.
  
- Browse through the Menu page to explore products.

- Click “Add to Cart” to add items dynamically.

- Use the Cart section to view, update, or delete items.

- Click “Buy Now” to simulate the checkout process.
